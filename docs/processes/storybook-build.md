# storybook-start

-   Alias: `sb-build`
-   Storybook is being setup to use `mhy`'s `Webpack` and `SWC` config.
-   Builds a standalone version of your Storybook to the `.sb` directory.

### Default command

```bash
build-storybook
```

### CLI Options

https://storybook.js.org/docs/configurations/cli-options/#for-build-storybook

### Example

```bash
mhy sb-build
# or
mhy storybook-build
```
