# node

Run scripts in Node.JS.
- TypeScript support.
- Uses `swc-register`
- Support for aliases.
- Default: `./src`
- Resolves `.js`, `.jsx`, `.ts` and `.tsx` files.

## Default command
```bash
node ./src
```

## CLI Options
https://nodejs.org/api/cli.html

## Examples

### Default CLI
```bash
mhy node
```

### Run a specific file/folder
```bash
mhy node file
```
